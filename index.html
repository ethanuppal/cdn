Syntax Highlighting

Load the following files on your webpage.

  • lib/syntax.js
  • lib/syntax.css
  • lib/autosyntax.js (optional, but recommended)

Then, indicate which <pre> tags should be highlighted by adding code to their class. In addition, add the specific language that the block should be highlighted in, e.g., code-c. If the autosyntax file is not included, you must have manual calls to highlight each block.

Example

<!-- in your <head> -->
<link rel="stylesheet" href="https://ethanuppal.github.io/cdn/lib/syntax.css">
<script src="https://ethanuppal.github.io/cdn/lib/syntax.js"></script>
<script src="https://ethanuppal.github.io/cdn/lib/autosyntax.js"></script>

<!-- in your <body> -->
<pre class="code code-c">
#include <stdio.h>
int main() {
    printf("Hello, world\n");
}
</pre>

The above code produces:

#include <stdio.h>
int main() {
    printf("Hello, world\n");
}

Footnotes

Load the library in your <head>.

<script src="https://ethanuppal.github.io/cdn/lib/footnotes.min.js"></script>

Set up a location on the page where footnotes will be placed by giving it the id citation-list.

<ol id="citation-list" />

Then create inline footnotes using a special <span> element.

<span data-cite
    n="1"
    text="This is some text."
    href="https://www.example.com" rel="nofollow" target="_blank"></span>

Only data-cite and either of text or href are necessary. To supply a custom number for the footnote, set n (the default is a basic counter). You can set rel and target to modify the properties of the resulting link when supplying href.

To access the footnotes, simply inspect the array FootnoteUserElement.list from javascript.

Unminified source file:

  • lib/footnotes.js

Image Overlay

Load the library in your <head>.

<script src="https://ethanuppal.github.io/cdn/lib/overlay.min.js"></script>

Add overlay-trigger to the class list of a div that contains images. The library will present the first child element that is an image or has the class overlay-image.

<div class="overlay-trigger">
    <img src="willbepresented.png" alt="Testing 123">
    <img src="image2.jpg" alt="Hello">
</div>

Finally, load the library with custom styling. Each color must be a valid CSS value for a color attribute.

// This is an abysmal color scheme intended for purposes of example only.
window.addEventListener('load', function() {
    ImageOverlay.load({
        backgroundColor: '#0000FF',
        borderColor: 'red',
        borderWidth: '10px',
        imageBorderColor: 'green',
        imageBorderWidth: '5px',
        persists: true
    });
});

persists refers to whether refreshing or leaving and exiting the page will automatically dismiss the popup. Default is false, but if true, it will remember that you opened it.

As a demo, this is Photo 51, the famous picture of the DNA double helix. The file is taken from Stanford's website. Try clicking on it!
